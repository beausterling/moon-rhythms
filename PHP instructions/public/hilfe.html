Die Swiss Ephemeris PHP Klasse

Version: 2.08 - Git Repro

Swetest berechnet einen vollständigen Satz von geozentrischen Planetenpositionen, für ein bestimmtes Datum oder eine Folge von Datumsangaben. Die Eingabe kann entweder ein Datum oder eine absolute julianische Tageszahl sein. 0:00 (Mitternacht).

Mit den richtigen Optionen kann der Swetest zur Ausgabe eines Ausdrucks verwendet werden Ephemeride und übertragen Sie die Daten in andere Programme wie Tabellenkalkulationen zur grafischen Darstellung.

Die Swiss Ephemeris PHP Klassen machen die einfache nutzung der swetest möglich um mit ihr die Planetenpositionen auf einem Server in PHP nutzen zu können. Sie sind also ein PHP-Adapter zur Konsole die auf die in C geschriebene Bibiliothek swetest zugreift.

Die PHP Klassen greifen auf die Konsole des Betriebsystem mit exec() zu und binden die Ephemeris Bibiliotheken ein. Die Ausgabe der Konsole machen sie dann wieder für PHP verfügbar. Wenn man möchte kann man die Ausgabe noch formatieren lassen:
  • PHP_ARRAY
  • JSON
  • PLAIN
